# Student Mental Health and Burnout Analysis Dashboard

Acest proiect reprezintă o aplicație web interactivă construită în **Python** folosind **Streamlit**. Scopul principal este explorarea și analizarea unui set de date extins (150.000 de înregistrări) privind sănătatea mintală și nivelul de epuizare (burnout) în rândul studenților.

## Despre Proiect

Viața academică modernă implică un nivel ridicat de stres, iar înțelegerea factorilor care declanșează efectul de burnout este esențială. Această aplicație transformă datele brute într-o poveste vizuală ușor de înțeles, permițând utilizatorilor să exploreze impactul factorilor academici, psihologici și de stil de viață.
